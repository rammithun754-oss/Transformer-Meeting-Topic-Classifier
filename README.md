# 🤖 Transformer Meeting Topic Classifier

A Transformer Encoder built from scratch using PyTorch to classify meeting-related sentences into business categories.

This project demonstrates the core architecture behind modern Transformer models including embeddings, positional encoding, self-attention, and transformer encoders.

---

# 🚀 Features

✅ Word Embeddings

✅ Positional Encoding

✅ Transformer Encoder

✅ Multi-Head Self Attention

✅ Sequence Classification

✅ Business Topic Classification

✅ PyTorch Implementation

---

# 🧠 Categories

The model classifies sentences into:

- Hiring
- Budget
- Action
- Decision
- Infrastructure
